# v1.1.0 需求文档

## 需求来源
kyo，2026-03-12 15:52

## 需求描述
用户端首页无法让新用户理解系统的使用方式，需要优化。

## 优化目标
1. 新增使用引导，让用户知道"这是什么、怎么用"
2. 按推荐学习路径排列功能入口
3. 显示学习进度概览，增加使用动力
4. 所有图标使用 SVG（Lucide），禁止使用 emoji

## 实现内容

### 新增
- `/api/client/stats` 接口 — 返回学习统计数据（知识点掌握数、对练次数、今日练习数等）
- `getHomeStats` API client 方法和 `HomeStats` 类型

### 修改
- **客户端首页** (`apps/client/src/app/(main)/page.tsx`)：
  - 顶部欢迎区保持不变
  - 新增学习进度概览（三张渐变卡片：知识点/对练/今日）
  - 功能入口改为 2x2 学习路径网格
  - 底部快捷入口（练习记录/我的档案）
